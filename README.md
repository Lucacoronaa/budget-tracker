# Budget Tracker

A mobile-first financial tracker built with FastAPI, PostgreSQL, Next.js, React and Tailwind CSS.

## Features

- User registration and login
- JWT authentication
- Personal income and expense tracking
- Dashboard summary
- Transaction list
- Add new transactions
- Macro and sub categories

## Tech Stack

### Backend
- Python
- FastAPI
- PostgreSQL
- Neon
- JWT authentication

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
